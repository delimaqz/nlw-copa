## 🚀| Instalação e execução
##### Clonar repositório
	git clone git@github.com:delimaqz/nlw-copa.git
  
##### Mobile
	npm install  
  
	npx expo start --clear 

##### Server
	npm install 
  
	npm run dev 
  
  

##### Web
	npm install 
  
	npm run dev
